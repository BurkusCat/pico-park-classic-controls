# Two Player Keyboard Config
This is the default configuration that generates for the game.

### Player 1 controls
**LEFT** - LEFT ARROW
**RIGHT** - RIGHT ARROW
**DOWN** - DOWN ARROW
**JUMP** - SPACEBAR
**ACTION** - SPACEBAR

### Player 2 controls
**LEFT** - A
**RIGHT** - D
**DOWN** - S
**JUMP** - W
**ACTION** - W
